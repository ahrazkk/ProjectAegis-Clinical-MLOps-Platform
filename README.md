# Aegis-Clinical-MLOps-Platform
Project Aegis: A GKE-orchestrated MLOps platform for real-time DDI relation extraction using a calibrated PubMedBERT model. High-availability (3-replica) microservice platform with REST/gRPC API contracts. Its 90% F1-score model is deployed on a cost-optimized, scale-to-zero GKE GPU node pool.
